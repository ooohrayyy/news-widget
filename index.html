Новостной виджет на JavaScript

Это небольшой виджет новостной ленты на чистом JavaScript, который меня попросили сделать в качестве тестового задания.

Внизу — кнопки управления, которыми можно настроить источник для получения новостей. Первые два источника находятся в файлах проекта, третий получает новости через GNewsAPI.
