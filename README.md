# Новостной виджет на JavaScript

Это небольшой виджет новостной ленты на чистом JavaScript, который меня попросили сделать в качестве тестового задания.

* [Ссылка на страницу проекта на GitHub Pages](http://ooohrayyy.github.io/news-widget)

## Функциональность

Виджет расположен в нижнем правом углу страницы — в него подгружается лента новостей из выбранного источника.

Первые два источника находятся в файле `sourceExample.js` в каталоге `resources`. Третий источник — [GNewsAPI](https://gnewsapi.net/).

## Технологии

Проект написан на чистом JavaScript, элементы стилизованы при помощи CSS. Для запросов используется Fetch API.

Версия без модулей хранится в [отдельной ветке](https://github.com/ooohrayyy/news-widget/tree/no-modules).
